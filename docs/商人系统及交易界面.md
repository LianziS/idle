# 商人系统及交易界面

> 版本：v1.0  
> 创建日期：2026-03-15  
> 状态：待开发

---

## 需求概述

在游戏侧边栏新增"商人"入口，不需要经验条等级。点击后，主界面展示商人列表（暂时做四个商人，功能一致，分别为**建筑师、铸甲师、缝缀师、药剂师**）；点击具体商人后，右侧弹出交易界面，支持查看商品、多选仓库资源并快速出售。

---

## 详细逻辑与布局

### 1. 侧边栏入口

在侧边栏功能菜单中增加"商人"图标及文字按钮。

**交互：** 点击后，主容器（侧边栏右侧的内容区）切换为商人列表界面。

---

### 2. 主容器：商人列表界面

**布局：** 以网格或列表形式展示多位商人（初期可先配置 2-3 位）。

**每个商人的卡片应包含：**

| 元素 | 描述 |
|------|------|
| 头像 | 显示商人头像图片 |
| 名字 | 显示商人名称 |
| 好感度进度条 | 显示当前好感度数值及等级（如：友好/中立），并配有进度条可视化 |

**点击事件：** 点击该商人的任意区域，页面右侧滑出交易弹窗。

---

### 3. 右侧交易弹窗

**触发方式：** 点击商人列表中的某位商人后，从屏幕右侧滑出。

**布局分为上下两部分：**

#### 3.1 上半部分：交易栏以及任务栏

**交易栏**（二级菜单栏的第一个）：
- 显示该商人当前出售的商品列表
- 每个商品项包含：图标、名称、价格（消耗哪种资源及数量）、购买按钮
- **暂定商品：** 四位商人分别出售：
  - 建筑师 → "建筑大师卷轴"（1000 元）
  - 铸甲师 → "锻造大师卷轴"（1000 元）
  - 缝缀师 → "裁缝大师卷轴"（1000 元）
  - 药剂师 → "药剂大师卷轴"（1000 元）

**交互：** 点击"购买"，检查资源是否足够，足够则完成交易并刷新仓库显示。

**任务栏**（二级菜单栏的第二个）：
- 显示该商人当前可以接取的任务
- 点击"任务"栏，下面的内容是横框，包含：任务名、任务内容、任务奖励、"领取任务"按钮
- 点击即可领取任务。此任务整体变为灰色，"领取任务"按钮变成"提交"
- 玩家在下方的仓库多选资源后点击提交就可以完成任务

#### 3.2 下半部分：我的仓库（用于出售）

**功能描述：** 显示玩家当前拥有的所有资源（如：木材、石材等）。

**新增多选卖出机制：**

1. 在仓库区域顶部，新增一个 **"选择"按钮**
2. 点击"选择"后，仓库中的每个资源玩家可以点击，单个、多个资源进行多选（选中项在资源框右下角打勾✓，绿色较短打勾）
3. 仓库区域底部出现 **"出售"按钮**，显示当前选中资源可卖出的总价
4. 点击"出售"后按钮内文本变为"确认出售"，再次点击才真正出售
5. 选中的资源从仓库移除，并获得相应货币

## 数据与逻辑要点

### 商人数据

需配置商人 ID、名称、头像、初始好感度、商品列表（商品 ID，价格）。

### 好感度系统

- 在商人页面的右上角显示
- 默认基础值为 `0.00`
- 显示为"图标" + "值"

### 快速多选卖出逻辑

1. 选择模式开启后，点击资源项即加入"待出售列表"
2. 支持再次点击取消选中
3. 卖出时统一结算
